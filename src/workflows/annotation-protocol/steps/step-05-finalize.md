### 5.2 Pilot Test

Before full-scale annotation:

**Pilot Parameters:**
- Sample size: 50-100 cases
- Annotators: 2-3 certified annotators
- Duration: 1-2 weeks

**Pilot Objectives:**
- [ ] Validate guidelines clarity
- [ ] Measure actual IAA
- [ ] Identify guideline gaps
- [ ] Estimate annotation throughput
- [ ] Test quality control processes

**Pilot Success Criteria:**
- IAA meets threshold
- < 5% cases require guideline clarification
- Annotators report guidelines are clear
- No systematic errors identified

### 5.3 Refine Based on Pilot

Document and address:
- Guideline ambiguities identified
- Edge cases not covered
- Tool/workflow issues
- Training gaps

Update protocol version after refinements.
