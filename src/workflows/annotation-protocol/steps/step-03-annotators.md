### 3.2 Training Program

**Phase 1: Orientation**
- Project overview and objectives
- Importance of annotation quality
- Regulatory context

**Phase 2: Technical Training**
- Annotation tool training
- Guidelines walkthrough
- Practice on sample cases (non-study data)

**Phase 3: Calibration**
- Annotate calibration set (N=20-50)
- Compare to gold standard
- Review and discuss discrepancies
- Re-calibrate if needed

### 3.3 Certification Criteria

Annotators must demonstrate:

| Metric | Threshold | Assessment Method |
|--------|-----------|-------------------|
| Agreement with gold standard | ≥ [X]% | Calibration set |
| Inter-annotator agreement | κ ≥ [0.X] | Paired annotation |
| Completion of training | 100% | Training log |
| Guidelines comprehension | Pass quiz | Written test |

### 3.4 Ongoing Qualification

- Periodic re-calibration (e.g., quarterly)
- Performance monitoring
- Remedial training if drift detected
- Documentation of all training activities

### 3.5 Annotator Registry

Maintain registry:

```markdown
| Annotator ID | Qualifications | Training Date | Certification Status | Last Calibration |
|--------------|----------------|---------------|---------------------|------------------|
| ANN-001 | [credentials] | [date] | Certified | [date] |
```
